## 🚀 Current Projects & Research

This repository is actively maintained, with new financial models and academic reports being added regularly. 

### Interactive Financial Models
* **Portfolio Optimization Model (*In Progress*):** A full-stack application built using Modern Portfolio Theory (MPT) to find optimal asset allocation. It calculates the Efficient Frontier, Max Sharpe Ratio, and Minimum Variance portfolios.
* **Payback Period Calculator:** An interactive tool to calculate standard and discounted payback periods for any given cash flow stream.

### Academic Research & Reports
* **Fundamental Analysis (Apollo Hospitals):** An in-depth security analysis covering qualitative factors, Porter's 5 Forces, industry trends, and future growth drivers for Apollo Hospital Enterprise Limited.
* **Financial Statement Analysis (Tata Motors):** Comprehensive evaluation using Income Statements, Balance Sheets, and detailed Ratio Analysis to assess financial health.

## 🛠️ Tech Stack

**Frontend (UI & Interactivity):**
* HTML5 & CSS3 (Custom styling, responsive grid layouts)
* Vanilla JavaScript (DOM manipulation, asynchronous API fetching)
* Chart.js (Data visualization & scatter plots)

**Backend (Quantitative Engine):**
* Python 3.x
* Flask & Flask-CORS (REST API infrastructure)
* `yfinance` & Financial APIs (Live market data ingestion)
* `scipy.optimize` (SLSQP minimization for portfolio weights)
* `pandas` & `numpy` (Financial data cleaning and matrix mathematics)

**Financial Modeling:**
* Advanced Excel & VBA
